**Python Automation Scripts for Security Scanning Workflows**

I have designed a modular, professional Python automation framework for orchestrating security scanning workflows on Kali Linux (including WSL environments). The design adheres strictly to PEP 8 guidelines, emphasizes security, error handling, configurability, and logging. It uses `subprocess` for reliable tool invocation, supports sequential and selective execution, and generates consolidated reports.

### Framework Architecture

- **scanner_orchestrator.py**: Main entry point for workflow management.
- **scanners/**: Package containing individual scanner modules (e.g., `nmap_scanner.py`, `nikto_scanner.py`).
- **config.py**: Centralized configuration (targets, output directory, options).
- **utils.py**: Shared utilities for logging, output parsing, and reporting.
- **requirements.txt**: Dependencies (minimal: `python-dotenv` for configuration flexibility).

The framework allows running full workflows or specific tools against one or more targets.

### Installation and Setup

1. Create a project directory (e.g., `security-scanner-automation`).

3. Install dependencies:

#### 2. `utils.py`
```python
"""Utility functions for logging, execution, and reporting."""

import logging
import subprocess
from datetime import datetime
from pathlib import Path
from typing import Dict, List, Optional

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s - %(name)s - %(levelname)s - %(message)s",
    handlers=[
        logging.FileHandler("scanner.log"),
        logging.StreamHandler()
    ]
)
logger = logging.getLogger(__name__)

def run_command(cmd: List[str], timeout: int = 300, cwd: Optional[Path] = None) -> subprocess.CompletedProcess:
    """Execute a shell command safely with timeout and error handling."""
    try:
        logger.info(f"Executing: {' '.join(cmd)}")
        result = subprocess.run(
            cmd,
            capture_output=True,
            text=True,
            timeout=timeout,
            cwd=cwd
        )
        if result.returncode != 0:
            logger.warning(f"Command failed with code {result.returncode}: {result.stderr}")
        return result
    except subprocess.TimeoutExpired:
        logger.error(f"Command timed out after {timeout} seconds")
        raise
    except Exception as e:
        logger.error(f"Execution error: {e}")
        raise

def save_report(content: str, target: str, tool: str) -> Path:
    """Save scan output to a timestamped report file."""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    report_path = Path(OUTPUT_DIR) / f"{target}_{tool}_{timestamp}.txt"
    report_path.write_text(content, encoding="utf-8")
    logger.info(f"Report saved: {report_path}")
    return report_path
```

#### 3. Example Scanner Module (`scanners/nmap_scanner.py`)
```python
"""Nmap scanner module."""

from pathlib import Path
from ..utils import run_command, save_report
from ..config import TOOL_CONFIG

def run_nmap(target: str) -> str:
    """Run Nmap scan against the target."""
    config = TOOL_CONFIG["nmap"]
    cmd = ["nmap"] + config["flags"].split() + [target]
    result = run_command(cmd, timeout=config["timeout"])
    output = f"=== Nmap Results for {target} ===\n{result.stdout}\n{result.stderr}"
    save_report(output, target, "nmap")
    return output
```
